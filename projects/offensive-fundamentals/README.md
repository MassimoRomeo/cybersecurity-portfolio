# Offensive Fundamentals

## Scenario
This project is based on beginner hands-on labs from Hack The Box and TryHackMe focused on enumeration, service discovery, and introductory exploitation.

## Objective
The goal was to build practical understanding of how beginner attack paths work in controlled lab environments.

## Tools Used
- Linux terminal
- Basic enumeration techniques
- Web and service analysis concepts

## What I Did
- Worked on beginner and very easy lab machines
- Practiced identifying exposed services and possible entry points
- Improved my understanding of basic attack methodology
- Focused on documenting the logic behind each step

## Outcome
This project helped me develop a stronger foundation in offensive security basics and technical reasoning.

## Skills Demonstrated
- Enumeration
- Service discovery
- Beginner penetration testing
- Technical documentation
